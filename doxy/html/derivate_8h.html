Target Information.

Version
0.3

This file provides information about the target, such as the architecture or the derivate.
The goal is to provide the same environment on different platforms, where the target specific information only needs to be changed in this file.

Changelog:

  • 0.3 12.01.17 Barth
    • Added Espressif ESP32
    • changed TARGET to ARCHITECTURE
  • 0.2 02.07.16 Barth
    • Introduced the concept of TARGET_ID to allow a flexible and easy configuration-set selection.
    • Added AURIX frequency definitions and timing constraints from the TC29xB Data Sheet V 1.1
